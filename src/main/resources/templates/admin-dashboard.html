Thêm Sản phẩm Mới

Sử dụng GraphQL mutation để thêm sản phẩm mới:

mutation {
  createProduct(
    title: "Tên sản phẩm"
    quantity: 10
    desc: "Mô tả sản phẩm"
    price: 1000000
    userId: 1
    categoryId: 1
  ) {
    id
    title
    price
  }
}
Mở GraphiQL
Thêm Danh mục Mới

Sử dụng GraphQL mutation để thêm danh mục mới:

mutation {
  createCategory(
    name: "Tên danh mục"
    images: "category.jpg"
  ) {
    id
    name
    images
  }
}
